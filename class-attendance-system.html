Class Attendance System

The Class Attendance System is a lightweight tool created to efficiently manage student attendance in an academic setting. Developed using C++ and Excel, this system automates the recording and reporting of attendance data, reducing manual effort and minimizing errors.

Project Highlights:

  • Role: Lead Developer
  • Technologies Used: C++, Excel
  • Features:
    • Automatic data entry and storage in Excel sheets.
    • Simple C++ interface for inputting and updating attendance records.
    • Customizable reporting tools for generating attendance summaries and detailed reports.
    • Support for multiple classes, subjects, and periods.
  • Outcome: Improved accuracy and efficiency in managing attendance records, praised by educators for its simplicity and reliability.

This project exemplifies my ability to develop practical software solutions with minimal resources while ensuring reliability and ease of use for end-users.

The system was designed to be user-friendly, with an intuitive interface built in C++ and seamless integration with Excel for data storage and reporting. This approach ensured that the system could be easily adopted without requiring significant changes to existing workflows.
